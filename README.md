# A terminal tool that uses AI to help you execute commands and debug problems that might appear in the way.

# How to use? 
- just type terminal-ai in the terminal and enjoy :)

# Installation
1. clone the repo
2. execute command ```pip install .```

# About the tools used
- LLM is used from your API key for openrouter.ai
- searching is done using your API key for firecrawl
- For both you can find some free stuff (not too limited free stuff btw :) )

# About the project 
- This project was implemented using Langgraph
- The code automatically detects the OS type, version, distro...etc. so you don't have to do it yourself.
- Please don't think of this project as a competitive to Large projects from industrial companies; it was done mainly for my own learning.
- also note it was done in 2 days (due to me being busy with other deep learning projects) so don't expect (for now) a lot of fancy features :)
- and btw, I only used the AI for enhancing visuals. The core was my pure human code :)
